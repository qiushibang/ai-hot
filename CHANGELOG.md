# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-02

### Added

- 多平台 AI 热点聚合：GitHub、X/Twitter、YouTube、Hugging Face
- Observatory 深空观测台风格深色主题仪表盘
- 主题搜索：输入自定义关键词精准抓取 AI 相关内容
- X 账号追踪：配置关注的 X 账号，按账号抓取最新动态
- 收藏箱：收藏感兴趣的内容，支持从收藏箱一键推送
- 飞书推送：支持 Webhook 和开放平台 API 两种方式
- 企业微信推送：支持机器人 Webhook 推送到群聊
- 推送记录：查看历史推送记录
- Chrome 新标签页替换：每次打开新标签页都是 AI 资讯看板
- 浅色/深色双模式切换
- 关键词过滤：支持包含/排除关键词
